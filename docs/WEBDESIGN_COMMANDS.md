# Web Design Team Command Protocol

아래 명령으로 메인 에이전트가 역할별 병렬 라운드를 수행한다.

## 1) 전략 라운드
- `웹디자인팀 라운드: <주제>`
- 예) `웹디자인팀 라운드: 메인 랜딩의 첫 전환율 올리기`

출력:
- 역할별 의견 요약
- 통합 실행안 1개

## 2) 카피 집중
- `웹카피 라운드: <페이지/섹션>`
- 예) `웹카피 라운드: Hero 섹션 CTA 개선`

## 3) 모바일 최적화 집중
- `웹모바일 QA: <페이지>`
- 예) `웹모바일 QA: /mbti`

출력:
- 깨짐/겹침/터치 타겟 이슈
- 수정 우선순위

## 4) 실험 설계
- `웹실험 설계: <목표 KPI>`
- 예) `웹실험 설계: CTA CTR +20%`

출력:
- A안/B안 가설
- 지표/기간/샘플 기준
- 승자 판정 규칙
